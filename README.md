# Egypro OneHub 2.0

Egypro OneHub is a modular internal operations platform for HR, payroll, employee self-service, warehouse management, projects, field updates, and project cash accountability.

## Local development

1. Copy `.env.example` to `.env.local` and add the public configuration for your Supabase project.
2. Install dependencies with `npm install`.
3. Start the application with `npm run dev`.

Run the complete local quality gate with `npm run verify`.

## Security

Never add a Supabase service-role key or another secret to a `VITE_*` variable. Browser configuration is public by design.
